leetcode_67_Add Binary

Given two binary strings, return their sum (also a binary string).

For example,

a = "11"

b = "1"

Return "100".

问题分析

将两个二进制字符串相加

解题思路

从字符串的低位到高位，每一位相加，然后再取余，并且注意计算进位carry。
