leetcode-303 Range Sum Query

问题描述

Given an integer array nums, find the sum of the elements between indices i and j (i ≤ j), inclusive.
Example:

Given nums = [-2, 0, 3, -5, 2, -1]
sumRange(0, 2) -> 1
sumRange(2, 5) -> -1
sumRange(0, 5) -> -3

问题分析

由于sumRange会多次调用，那么我们就应该尽量降低它的时间复杂度，同时由于构造函数只有一次，那么他的时间复杂度稍微高一些也可以。
因此，我将NumArray初始化的操作计算为S[n]=a[0]+..a[n]，前n项和。在sumRange中计算a[n]+...+a[m]=S[m]-S[n-1]。
总结起来这道题就是一道数列的前n项和与第n项之间的关系。
