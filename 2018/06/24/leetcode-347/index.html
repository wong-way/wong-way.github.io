leetcode No347. Top K Frequent Elements(bucket sort)

leetcode No347. Top K Frequent Elements

问题描述

Given a non-empty array of integers, return the k most frequent elements.

问题分析

这道题我得思路是，首先统计每个数字出现的频率，得到key-value这样的键值对，然后将value排序，从高到低获取key，中间要用到键值反转，同时这里可以使用桶排序来解决这个问题

解题代码
